# astrbot_plugin_bilibili

B站视频搜索爬虫插件 for AstrBot

## 功能特点

- 🔍 按关键词搜索B站视频
- 📅 按发布时间排序（最新优先）
- ▶️ 获取播放量、弹幕数、发布时间
- 🔥 支持播放量/小时筛选
- ⚙️ 支持Web界面配置

## 安装方法

1. 在 AstrBot 插件市场中搜索 `b站爬虫` 直接安装
2. 或通过 WebUI 从链接安装: `https://github.com/your-repo/astrbot_plugin_bilibili`

## 配置说明

在插件配置页面配置以下选项：

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| sessdata | string | - | B站 SESSDATA Cookie（推荐配置，可绕过反爬） |
| default_keyword | string | 杀戮尖塔2 | 默认搜索关键词 |
| target_count | int | 10 | 目标数量 |
| min_play_per_hour | float | 1000.0 | 播放量/小时筛选阈值 |
| use_collect_mode | bool | true | 集满模式开关 |
| default_max_count | int | 50 | 普通模式最大获取数量 |
| export_json | bool | false | 是否导出JSON |

## 使用方法

### 指令列表

| 指令 | 说明 | 示例 |
|------|------|------|
| `/b站搜索` | 搜索视频 | `/b站搜索 Python教程` |
| `/b站搜索 <关键词>` | 指定关键词搜索 | `/b站搜索 原神 20` |
| `/b站搜索 <关键词> <数量>` | 指定关键词和数量 | `/b站搜索 杀戮尖塔2 25 1000` |
| `/b站搜索 <关键词> <数量> <阈值>` | 完整参数搜索 | `/b站搜索 艾尔登法环 10 2000` |
| `/b站热门` | 使用默认配置搜索 | `/b站热门` |
| `/b站配置` | 查看当前配置 | `/b站配置` |

### 参数说明

- `<关键词>`: 搜索关键词
- `<数量>`: 目标数量（集满模式）或最大获取数量（普通模式）
- `<阈值>`: 播放量/小时筛选阈值，只展示大于此值的视频

### 模式说明

**集满模式**（默认）:
- 持续搜索直到满足筛选条件的视频达到目标数量
- 适合获取高质量热门视频

**普通模式**:
- 先获取固定数量的视频
- 然后按阈值筛选
- 适合需要快速获取结果

## 获取 SESSDATA

1. 登录 B站 (https://www.bilibili.com)
2. 按 `F12` 打开开发者工具
3. 切换到 `Application` 标签
4. 左侧找到 `Cookies` → `https://www.bilibili.com`
5. 复制 `SESSDATA` 的值
